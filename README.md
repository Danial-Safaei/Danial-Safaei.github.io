# Danial Safaei

Personal website: https://danial-safaei.github.io

PhD researcher at WMG, University of Warwick (Safe Autonomy Research Group). Research interests: Safe AI and trustworthy evaluation for autonomous systems, including synthetic-data fidelity and explainable evaluation.

## Structure

- `index.html` — homepage (about, journey, focus, publications, code, contact).
- `blog.html` + `blog.js` — dependency-free Markdown blog.
- `posts/` — blog posts as Markdown, listed in `posts/posts.json`.
- `style.css` — styling, theming, and the site watermark.
- `interact.js` — shared client-side behaviour.

## Adding a blog post

1. Create `posts/<slug>.md` (plain Markdown).
2. Add an entry to `posts/posts.json` with `slug`, `title`, `date` (YYYY-MM-DD), `summary`, and `tags`.
3. Commit and push — GitHub Pages deploys automatically. No build step.
